# Otaku's RPG - Sistema de Fichas

Sistema web para criação e gerenciamento de fichas de personagens de RPG, desenvolvido para a comunidade Otaku's RPG.

## Funcionalidades

- Criação de fichas de personagem
- Listagem de personagens criados
- Salvamento automático no navegador
- Interface moderna e responsiva
- Sistema completo de atributos

## Tecnologias

- [React](https://react.dev/) - Biblioteca para construção de interfaces
- [Vite](https://vitejs.dev/) - Build tool e servidor de desenvolvimento
- [Styled Components](https://styled-components.com/) - CSS-in-JS para estilização
- [React Router DOM](https://reactrouter.com/) - Roteamento da aplicação

## Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

- [Node.js](https://nodejs.org/) (versão 18.0 ou superior)
- [npm](https://www.npmjs.com/) (gerenciador de pacotes do Node)
